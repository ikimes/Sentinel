## Implementation Steps

1. Add one narrow smoke script under the canonical diagnostics layout.
2. Submit a real compliance request against the running stack.
3. Wait until the request reaches a readable state.
4. Assert `GET /api/compliance/check/recent` returns the request.
5. Assert `GET /api/compliance/check/{requestId}/history` returns the request timeline.
6. Assert that full request content is not exposed in these responses.
7. Keep the script minimal and readable for future agents.
