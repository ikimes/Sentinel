# Sentinel Archived Docs

This directory keeps historical working notes, analysis memos, and completed refinement artifacts that are still useful for traceability but should not clutter the active docs tree.

Use this archive for documents that are:

- valuable as process history
- no longer canonical
- superseded by later implementation or planning work

## Current Archive Buckets

- `analysis/`
  - historical project-readiness and verification-gap assessments
- `execution/`
  - completed execution boards, slice packets, proof checkpoints, and implementation planning notes
- `ui/`
  - completed inspection and refinement notes for the early Blazor shell

## Archiving Rule

Archive rather than delete when a document still helps explain:

- why a decision was made
- what was learned during execution
- how the project evolved

Active roadmap, implementation contracts, and package docs should remain outside this directory.
